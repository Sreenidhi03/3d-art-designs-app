# 3D Art Relief Design App

This is a web-based application for converting 2D images into 3D relief designs using height mapping techniques. The app is built using Python and Streamlit.

## Features

- Upload 2D images in formats like JPG, PNG, BMP.
- Convert images to grayscale height maps.
- Generate interactive 3D STL previews.
- Export STL and G-code files for 3D printing/CNC.
- Batch processing of multiple images.
- Automated PDF report generation.

## Requirements

- Python 3.8 or higher
- Streamlit
- NumPy
- Pillow
- Matplotlib
- OpenCV
- Trimesh
- Plotly
- numpy-stl
- ReportLab

## How to Run

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Run the Streamlit app:
   ```
   streamlit run art3.py
   ```

3. Open the app in your browser and start uploading images!
